# Discord Community

Back-end part

## Описание

Продолжение истории о дипломной работе, а если быть точнее - её бэкенд.

* Функционал: осуществление запросов на локальный SQL-сервер для последующей отправки данных на фронтенд
* Стек: C#, .NET 5
